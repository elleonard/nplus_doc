というわけで、Hexo導入にあたってのメモを書きます。

引っかかったこと

同名カテゴリが複数現れる

  • Hexoのカテゴリは階層構造を持つ
  • つまり、「ゲーム」カテゴリ内の「艦これ」カテゴリと、「艦これ」カテゴリ内の「ゲーム」カテゴリが明確に区別される
  • 面倒だが、カテゴリを複数設定する場合は順番を揃えて設定する
  • また、一度間違った階層でgenerateしてしまったものは、 hexo clean で抹殺する
